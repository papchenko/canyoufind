## 🚀 About the Project

Canyoufind is an **interactive web platform** where users participate in quests to discover secret locations. Players need to use hints and intuition to find hidden spots, and after discovering the mystical sign, they can retrieve a secret code. The platform encourages engagement, exploration, and competitive fun.

**Key Features:**
- Time-limited quests — each secret location is available for a limited time.
- Hints system — helps users navigate and discover hidden places.
- Interactive design with immediate feedback upon completing a quest.
- Mobile-friendly and responsive UI for seamless experience on any device.

---

## 🛠️ Technologies Used

This project leverages **modern web technologies**:

- **Frontend:** React, JavaScript (ES6+), Bootstrap, HTML5, CSS3
- **State Management:** Redux
- **Routing & Tools:** React Router, Vite
- **Backend / Realtime Database:** Firebase
- **Deployment:** GitHub Pages
- **Other Tools:** Axios for API requests, Toastify for notifications, Git & GitHub for version control

## ⚡ How It Works

1. Choose a quest to find a secret location.
2. Use the provided hints to locate the spot before time runs out.
3. Once the mystical sign is discovered, click it to get the secret code.
4. If the time expires, the resource becomes temporarily unavailable.
